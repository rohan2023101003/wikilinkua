WikiLinkua Privacy Policy

This page summarizes what the app stores, which external services it uses, and what permissions certain features need.

What we store

  • Selected known languages and target language
  • Word progress and phrase review state in the browser
  • Temporary session state for login and upload flows
  • Wikimedia username and id when login is enabled

What we do not store

  • Passwords
  • Payment details
  • A central database of learning progress in the current code path

External services

  • Wikidata Query Service
  • Tatoeba
  • Wikimedia MinT
  • Wikimedia Commons
  • Wikimedia OAuth
  • Wikipedia APIs

Permissions

  • Microphone access for pronunciation recording
  • Login access for identity-aware features
  • Commons upload permission for publishing audio
  • Network access to the services above

Retention and deletion

Local progress can be removed by clearing site data for the domain. If the project adds cross-device sync in the future, deletion and retention rules must be documented and enforced before launch.

This page is intentionally short because the current app does not yet have a central database; the main privacy risk today is third-party request exposure and browser-stored progress.
