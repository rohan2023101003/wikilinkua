WikiLinkua User Guide

Learn a new language through the languages you already know. This page is a quick in-site guide.

Documentation hub

  • Docs index
  • Privacy policy
  • Project on GitHub
  • Report issues / give feedback

How to start

  1. Pick the languages you already know.
  2. Pick the target language you want to learn.
  3. Use Home for mixed practice, Words for browsing, Phrases for sentence practice, and Progress to track your review state.

Your progress is stored in the browser on the device you are using.

Key terms in WikiLinkua

  • Bridge word: a target-language word connected to something you already know through meaning (and often sound).
  • Cognate: words related by origin across languages, marked from Wikidata cognate links.
  • Friend / Faux: lookalike words across languages that are either true matches (friend) or misleading mismatches (faux / false friend).
  • Phonetic match: words that sound similar enough to help memory even if spelling differs.
  • Gloss: the short meaning label shown for a word.
  • Leitner box: the spaced-repetition level used to decide when a word comes back for review.

What each practice mode means

  • Bridge flashcards: reveal target words and meanings quickly.
  • Friend or Faux: decide whether a similar-looking pair is a true match or a trap.
  • Recall / Reverse recall: answer from meaning to word, or word to meaning.
  • Odd one out / Match pairs: pattern drills to strengthen discrimination and recall.
  • Phrases: sentence-level practice built from words you already know or are learning.
  • Video / Article: learn from real Wikimedia media and text with subtitle or extract-based flashcards.

What the main labels mean

  • New / unseen: you have not answered the word yet.
  • Learning: you have started practicing the word, but it is not mastered yet.
  • Known: the word has crossed the mastery threshold.
  • Due for review: the word is ready to come back into practice.

A single correct answer does not usually make a word “known”; it normally moves the word up one step at a time.

How phrases work

Phrases are built from words you already know or are learning. When available, the app prefers human translations from Tatoeba; if none are available, it falls back to Wikimedia MinT.

Phrase review is separate from word review. That means a sentence can come back later even after you have seen it once.

Privacy and access

  • No password is stored by WikiLinkua.
  • Real login uses Wikimedia OAuth and only reads your Wikimedia identity.
  • Pronunciation uploads need Commons upload permission.
  • Learning progress currently stays in your browser unless a future sync layer is added.

Third-party services

  • Wikidata Query Service
  • Tatoeba
  • Wikimedia MinT
  • Wikimedia Commons
  • Wikipedia APIs
  • Wikimedia OAuth

UI style

The UI follows Wikimedia Codex style: blue primary actions, neutral gray cards and borders, clear spacing, and responsive layouts.

Future scope

  • Cross-device progress sync
  • Better explanations in the app
  • More detailed progress and streak views
  • Richer pronunciation practice
  • More language coverage and content sources
